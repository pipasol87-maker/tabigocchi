# ✈️ たびごっち — 旅する英会話アプリ 指示書

かわいいキャラを育てながら、**旅のシーン別に英会話フレーズ**を覚えるアプリ。
たんごっちIELTSと同じエンジン（SRS復習・✅覚えたチェック・3日後テスト・リスト化）をそのまま使えます。

**今回の旅程に合わせた構成**：ハンガリー（娘さんの大学）→ 電車でウィーン（美術館・歴史観光）→ ローマ（晩ごはん・スーパー・買い物）

## 新しいシーン（バージョン）の追加方法

`travel-words.js` の `window.TRAVEL_DECKS` 配列にデッキを1個追加するだけ：

```js
{id:"pharmacy", name:"💊 薬局", data:[
{w:"Do you have something for a headache?", pos:"p", jp:"頭痛に効くものはありますか",
 ex:"Do you have something for a headache? / Yes, this one works well.",
 exjp:"頭痛に効くものはありますか。/ はい、これがよく効きますよ。",
 col:"headache", coljp:"頭痛"},
// …このオブジェクトをフレーズの数だけ並べる（かんたん→応用の順）
]},
```

**フィールドの意味**
- `w` … 英語フレーズ（そのまま読み上げされる）
- `pos` … `"p"` 固定（フレーズ）
- `jp` … 日本語の意味（4択の選択肢になる）
- `ex` … 会話例「自分のセリフ / 相手の返事」（スラッシュ区切り・回答後に読み上げ）
- `exjp` … 会話例の和訳
- `col` / `coljp` … キーワードとその意味（回答後に🔗表示）

## AIに新シーンを作らせる指示書（コピペ用）

新しいバージョンが欲しいときは、ClaudeやChatGPTにこれを貼るだけ：

```
旅行英会話アプリ用のフレーズデータを作ってください。

【シーン】＝＝ここにシーンを書く（例：イタリアの薬局で買い物）＝＝
【フレーズ数】14個（かんたん→応用の順に並べる）

【出力形式】JavaScriptのオブジェクト配列。各要素は：
{w:"英語フレーズ", pos:"p", jp:"日本語の意味",
 ex:"自分のセリフ / 相手の返事（英語）", exjp:"その和訳（同じくスラッシュ区切り）",
 col:"フレーズ内のキーワード", coljp:"キーワードの意味"}

【条件】
- 実際の旅行者がそのまま口に出せる、短くて自然な表現にする
- ex は必ず「会話」（自分→相手の返事）にする
- jp は4択クイズの選択肢になるので、他と紛らわしくない訳にする
- 丁寧すぎず、カジュアルすぎない observed tone
```

できた配列を `travel-words.js` に `{id:"◯◯", name:"絵文字＋シーン名", data:[ここに貼る]},` として追加すれば完成。

## 使い方（たんごっちと同じ）

1. シーンピルで場面を選ぶ → 4択でフレーズの意味を当てる
2. 🔊自動発音ON（設定）なら、フレーズ→会話例の順に読み上げ。**旅行前の耳慣らしに最適**
3. もう言える表現は「✅ 覚えた」→ **3日後にテスト** → 合格で完全マスター👑
4. 📝テストタブ →「📋 ボタン一つでリスト化」で覚えた/覚えてないを一覧・コピーして紙のメモにも

**旅行前のおすすめ学習順**：✈️飛行機 → 🏨ホテル → 🍳朝ごはん →（滞在に合わせて）🚆電車 → 🖼️美術館 → 🍽️晩ごはん → 🛒買い物。🗣️現地語あいさつは移動中にサラッと。
